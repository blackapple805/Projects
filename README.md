# Projects

A collection of personal and professional software projects spanning web apps,
data/ML tools, automation, DevOps, and systems programming. Each folder is a
self-contained project with its own README where applicable.

Languages span JavaScript/React, Python, Go, and C++.

## Featured work

**Capstone — full-stack threat-detection platform (remote, distributed team).**
Worked across the full stack: React frontend (dashboard, threat-detection views,
auth flow), the Node/Express backend and API, and the threat-detection logic.
The team delivered the final in-person presentation in California while I
contributed remotely from Paris.

## Getting started

Each project is independent. To run one, `cd` into its folder and follow its own
README / install steps — typically:

```bash
# Node projects
npm install && npm start

# Python projects
pip install -r requirements.txt
```
